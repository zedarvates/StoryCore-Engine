# Changelog

All notable changes to the StoryCore API will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [v1.0.0] - 2024-01-15

### Added

- Initial release of StoryCore Complete API System (`*`)
- 113 endpoints across 14 functional categories (`*`)
- Async task management for long-running operations (`storycore.task.status`, `storycore.task.cancel`)
- Authentication and authorization support (`storycore.security.*`)
- Rate limiting and caching capabilities (`*`)
- OpenAPI 3.0 specification generation (`storycore.api.openapi`, `storycore.api.schema`)
