# Templates de Style pour la Génération de Prompts d'Images

Ce document présente des templates de style pour la génération de prompts d'images, basés sur les méthodes et techniques extraites du fichier "Comment créer de belles infographies.txt". Ces templates sont conçus pour faciliter la création d'infographies professionnelles et adaptées à divers besoins.

## Structure de Base d'un Prompt

Un prompt efficace pour générer une infographie doit inclure les éléments suivants :

1. **Sujet à traiter** : Précisez le sujet ou la sous-thématique de l'infographie.
2. **Disposition** : Choisissez le type de disposition (ex. : processus, carte mentale, tableau, etc.).
3. **Style graphique** : Sélectionnez un style visuel adapté (ex. : flat design, dashboard, comic book, etc.).
4. **Orientation** : Définissez l'orientation (paysage, portrait, carré).
5. **Niveau de détail** : Précisez le niveau de détail (concis, standard, détaillé).
6. **Exigences visuelles** : Incluez des critères tels que la hiérarchie typographique, la lisibilité, et l'espace négatif.

## Exemples de Templates

### Template 1 : Infographie Professionnelle

```
Créer une infographie professionnelle pour expliquer [sujet].
Structure : [disposition, ex. : processus en 5 à 7 étapes].
Format : [orientation, ex. : 16/9].
Style graphique : [style, ex. : dashboard entreprise].
Niveau de détail : standard.
Exigences visuelles :
- Hiérarchie typographique claire (titre, sous-titre, corps de texte).
- Maximum 7 éléments par section.
- Texte lisible sur écran.
- Icônes simples et cohérentes.
- Beaucoup d'espace négatif.
Générer une infographie mémorable et professionnelle, adaptée pour présentation, blog et réseaux sociaux.
```

### Template 2 : Infographie pour Enfants

```
Créer une infographie éducative pour expliquer [sujet] aux enfants.
Structure : [disposition, ex. : carte mentale].
Format : [orientation, ex. : carré].
Style graphique : [style, ex. : Lego 3D].
Niveau de détail : concis.
Exigences visuelles :
- Couleurs vives sur fond blanc.
- Effet plastique sur les personnages et les formes.
- Ombrage marqué.
- Texte simple et compréhensible.
Ne pas mettre de majuscules dans les titres, sauf au premier mot, aux noms propres et dans les acronymes.
```

### Template 3 : Infographie Technique

```
Créer une infographie technique pour expliquer [sujet] à un public expert.
Structure : [disposition, ex. : tableau de données].
Format : [orientation, ex. : 16/9].
Style graphique : [style, ex. : mode sombre].
Niveau de détail : détaillé.
Exigences visuelles :
- Palette de couleurs néon.
- Texte technique et précis.
- Maximum 7 éléments par section.
- Beaucoup d'espace négatif.
Avant de générer l'image, vérifier scrupuleusement l'orthographe du texte.
```

### Template 4 : Infographie Ludique

```
Créer une infographie ludique pour expliquer [sujet].
Structure : [disposition, ex. : chronologie].
Format : [orientation, ex. : 16/9].
Style graphique : [style, ex. : comic book].
Niveau de détail : standard.
Exigences visuelles :
- Couleurs vives et dynamiques.
- Effet de style années 70-80.
- Texte lisible et engageant.
- Icônes et illustrations cohérentes.
```

### Template 5 : Infographie Corporate

```
Créer une infographie corporate pour expliquer [sujet].
Structure : [disposition, ex. : dashboard KPI].
Format : [orientation, ex. : 16/9].
Style graphique : [style, ex. : infographie magazine].
Niveau de détail : standard.
Exigences visuelles :
- Palette de couleurs corporate (tons bleus et blancs).
- Fond noir avec deux couleurs pour mettre en évidence les aspects clés.
- Texte clair et net.
- Maximum 7 éléments par section.
```

## Bonnes Pratiques

1. **Précision du Sujet** : Toujours préciser le sujet ou la sous-thématique pour orienter l'infographie.
2. **Adaptation à l'Audience** : Ajuster le style et le niveau de détail en fonction de l'audience cible (enfants, experts, professionnels, etc.).
3. **Lisibilité** : Assurer que le texte est lisible et que les éléments visuels sont bien espacés.
4. **Correction Orthographique** : Utiliser des phrases comme "Avant de générer l'image, vérifier scrupuleusement l'orthographe du texte" pour réduire les erreurs.
5. **Expérimentation** : Tester différents styles et dispositions pour trouver le meilleur rendu.
