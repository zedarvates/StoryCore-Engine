# Tech Stack

## Frontend
The frontend is a desktop application built with Electron, providing a native Windows interface. It includes a web-based UI component in the `creative-studio-ui` directory, utilizing HTML, CSS, and JavaScript (managed via Node.js and npm for development and building).

## Backend
The backend is implemented in Python, featuring scripts for various functionalities including AI model handling, project management, and integrations. It leverages ComfyUI for stable diffusion-based image generation and processing, along with custom Python modules for logic, calculations, and API interactions.

## Database
Data storage is handled through JSON files, used for persisting project data, character information, audio exports, and configuration settings throughout the application.
