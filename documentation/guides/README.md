# Guides Utilisateur - StoryCore

Ce dossier contient des guides pratiques et tutoriels pour vous aider à utiliser efficacement StoryCore. Chaque guide est conçu pour être facile à suivre et applicable à des scénarios réels.

## Structure

- [Installation ComfyUI](COMFYUI_SETUP.md) - Guide complet pour installer et configurer ComfyUI
- [Guide Multi-Instance](MULTI_INSTANCE_GUIDE.md) - Configuration avancée avec plusieurs instances
- [Dépannage](TROUBLESHOOTING.md) - Résolution des problèmes courants

## Prérequis

Avant de commencer, assurez-vous d'avoir :

- ✅ StoryCore installé
- ✅ Comptes utilisateur configurés
- ✅ Accès internet stable
- ✅ Espace disque suffisant

## Comment Utiliser ces Guides

1. **Choisissez le guide approprié** selon votre besoin
2. **Lisez attentivement** les instructions
3. **Suivez étape par étape** les procédures
4. **Vérifiez les résultats** à chaque étape
5. **Consultez le support** si vous rencontrez des problèmes

## Support

Si vous avez des questions ou besoin d'aide :

- Consultez le [Guide de Dépannage](TROUBLESHOOTING.md)
- Vérifiez les [Notes de Version](../CHANGELOG.md)
- Contactez le support utilisateur : support@storycore.io
