## Introduction

Welcome to **StoryCore Creative Studio**, a professional video storyboard editor designed to help you plan and create cinematic sequences with guaranteed visual coherence. This application bridges the gap between creative vision and technical execution, providing an intuitive interface for video production planning.

### Key Features

- **Visual Storyboard Canvas**: Drag-and-drop interface for arranging shots
- **Professional Timeline Editor**: Precise control over timing and transitions
- **Advanced Audio Management**: Multi-track audio with surround sound support
- **AI-Powered Chat Assistant**: Natural language project creation
- **Real-time Preview**: See your storyboard come to life
- **StoryCore-Engine Integration**: Seamless backend processing

## Getting Started

### System Requirements

- **Operating System**: Windows 10+, macOS 10.15+, or Linux
- **RAM**: 8GB minimum, 16GB recommended
- **Storage**: 2GB free space
- **Browser**: Chrome 90+, Firefox 88+, Safari 14+, or Edge 90+

### Installation

1. Download the Creative Studio installer from the official website
2. Run the installer and follow the on-screen instructions
3. Launch the application from your applications folder

### First Launch

When you first open Creative Studio, you'll see the **Landing Page** with two main options:

- **Create New Project**: Start a fresh storyboard
- **Open Existing Project**: Continue working on a saved project

## Interface Overview

The Creative Studio interface consists of five main areas:

### 1. Menu Bar (Top)

- **File**: New, Open, Save, Export
- **Edit**: Undo, Redo, Cut, Copy, Paste
- **View**: Toggle panels, Zoom, Grid settings
- **Help**: Documentation, About

### 2. Asset Library (Left Panel)

Browse and manage reusable assets:
- Images
- Audio files
- Templates
- Transitions
- Effects
- Text templates

### 3. Storyboard Canvas (Center)

Visual workspace where you arrange shots in a grid layout. Each shot displays:
- Thumbnail image
- Title and description
- Duration
- Audio track indicators

### 4. Timeline (Bottom Center)

Temporal representation showing:
- Shot sequence
- Duration bars
- Transitions
- Audio waveforms
- Playhead position

### 5. Properties Panel / Chat Assistant (Right Panel)

Toggle between:
- **Properties Panel**: Edit selected shot details
- **Chat Assistant**: AI-powered natural language interface

## Creating Your First Project

### Step 1: Initialize Project

1. Click **"Create New Project"** on the landing page
2. Enter a project name
3. Choose project settings (resolution, frame rate)
4. Click **"Create"**

### Step 2: Add Your First Shot

**Method 1: Manual Creation**
1. Click the **"+"** button in the canvas
2. Fill in shot details:
   - Title
   - Description
   - Duration (in seconds)
3. Upload an image (optional)
4. Click **"Add Shot"**

**Method 2: Using Chat Assistant**
1. Click the chat icon in the top-right
2. Type: "Create a shot showing a sunrise over the city, 5 seconds"
3. The AI will generate the shot automatically

### Step 3: Arrange Shots

- **Drag and drop** shots to reorder them
- Shots **snap to grid** for precise alignment
- **Double-click** a shot to edit details

## Working with Shots

### Selecting Shots

- **Click** a shot to select it (blue border appears)
- **Shift+Click** to select multiple shots
- **Ctrl+A** to select all shots

### Editing Shot Properties

When a shot is selected, the Properties Panel displays:

1. **Basic Properties**
   - Title
   - Description
   - Duration

2. **Image**
   - Upload new image
   - Replace existing image
   - Remove image

3. **Audio Tracks**
   - Add multiple audio layers
   - Adjust volume, pan, fade
   - Apply audio effects

4. **Visual Effects**
   - Apply filters (vintage, blur, etc.)
   - Adjust parameters
   - Stack multiple effects

5. **Text Layers**
   - Add titles and captions
   - Customize font, size, color
   - Animate text

6. **Animations**
   - Add keyframes
   - Animate position, scale, rotation, opacity
   - Choose easing curves

### Deleting Shots

- Select shot(s) and press **Delete** key
- Or right-click and select **"Delete"**

## Managing Assets

- **Audio**: MP3, WAV, OGG
- **Templates**: Pre-configured shot templates
- **Transitions**: Fade, dissolve, wipe, slide

**Drag and Drop**:
1. Select an asset from the library

3. A new shot is created with the asset

- Use the search bar to find assets by name

## Timeline Editing

### Timeline Navigation

- **Zoom**: Use mouse wheel or zoom controls
- **Pan**: Click and drag the timeline
- **Playhead**: Click anywhere to jump to that time

### Reordering Shots

1. Click and drag a shot in the timeline
2. Drop it at the desired position
3. Other shots adjust automatically

### Adjusting Duration

1. Select a shot in the timeline
2. Drag the right edge to extend/shorten
3. Or enter precise duration in Properties Panel

### Timeline Markers

- **Current Time**: Red playhead line
- **Total Duration**: Displayed at the end
- **Time Markers**: Show seconds/minutes

## Audio Management

### Adding Audio Tracks

1. Select a shot
2. In Properties Panel, click **"Add Audio Track"**
3. Choose audio file from Asset Library or upload new

### Audio Controls

Each audio track has:
- **Volume**: 0-100%
- **Pan**: Left (-100) to Right (+100)
- **Fade In/Out**: Duration in seconds
- **Mute/Solo**: Toggle buttons

### Audio Effects

Available effects:
- **Limiter**: Prevent clipping
- **Voice Clarity**: Auto-enhance dialogue
- **Gain**: Adjust overall level (-60dB to +60dB)
- **Distortion**: Add grit (soft, hard, tube, fuzz)
- **Bass Boost**: Enhance low frequencies
- **Treble Boost**: Enhance high frequencies
- **EQ**: 3-band equalizer
- **Compressor**: Dynamic range control
- **Noise Reduction**: Remove background noise

### Audio Presets

Quick-apply professional settings:
- **Podcast**: Voice-optimized
- **Music Video**: Balanced mix
- **Cinematic**: Wide dynamic range
- **Dialogue**: Clear speech

### Surround Sound

1. Select audio track
2. Click **"Surround Sound"** tab
3. Choose mode: Stereo, 5.1, or 7.1
4. Adjust channel levels or use spatial positioner
5. **AI Suggestions**: Click "Ask AI" for scene-based presets

### Automation Curves

Animate audio parameters over time:
1. Select audio effect
2. Click **"Automation"** button
3. Add keyframes by clicking on the curve
4. Drag keyframes to adjust time and value
5. Choose interpolation: Linear, Smooth, Step, Bezier

### Voiceover Generation

1. Click **"Add Voiceover"** in Audio Panel
2. Enter text to speak
3. Select voice (male, female, neutral)
4. Choose language
5. Adjust speed and pitch
6. Select emotion (optional)
7. Click **"Generate"**
8. AI generates audio and adds to track

## Visual Effects

### Applying Effects

1. Select a shot
2. In Properties Panel, click **"Effects"** tab
3. Browse effect library
4. Click an effect to apply

### Effect Parameters

Each effect has adjustable parameters:
- **Intensity**: 0-100%
- **Specific controls**: Brightness, contrast, blur radius, etc.

### Effect Stacking

- Apply multiple effects to one shot
- Drag to reorder effects
- Effects apply in order from top to bottom

### Effect Presets

Common effect combinations:
- **Vintage**: Sepia tone + grain + vignette
- **Cinematic**: Color grade + letterbox
- **Dream**: Soft focus + glow
- **Noir**: High contrast + desaturation

## Text and Titles

### Adding Text Layers

1. Select a shot
2. Click **"Add Text Layer"** in Properties Panel
3. Enter text content
4. Customize appearance

### Text Properties

- **Font**: Choose from system fonts
- **Size**: Adjust font size
- **Color**: Text and background color
- **Position**: X/Y coordinates (percentage)
- **Alignment**: Left, center, right
- **Timing**: Start time and duration

### Text Styling

- **Bold**, *Italic*, Underline
- **Stroke**: Outline color and width
- **Shadow**: X/Y offset, blur, color

### Text Animation

Available animations:
- **Fade In/Out**: Smooth opacity transition
- **Slide In/Out**: Move from edge
- **Typewriter**: Character-by-character reveal
- **Bounce**: Spring effect

### Text Templates

Pre-designed title styles:
- **Lower Third**: Name and title
- **Opening Title**: Centered large text
- **End Credits**: Scrolling text
- **Subtitle**: Bottom-aligned caption

## Transitions

### Adding Transitions

1. Select two adjacent shots
2. Click **"Add Transition"** in Properties Panel
3. Choose transition type
4. Adjust duration

### Transition Types

- **Fade**: Gradual opacity change
- **Dissolve**: Cross-fade between shots
- **Wipe**: Directional reveal (left, right, up, down)
- **Slide**: Push one shot off screen
- **Zoom**: Scale transition

### Transition Settings

- **Duration**: 0.5 to 5 seconds
- **Direction**: For directional transitions
- **Easing**: Linear, ease-in, ease-out, ease-in-out

### Previewing Transitions

- Click **"Preview"** button
- Or use timeline playback to see transitions in context

## Exporting Projects

### Export Options

1. Click **File > Export**
2. Choose export format:
   - **StoryCore Project**: JSON format for backend processing
   - **PDF Storyboard**: Visual reference document
   - **Video Preview**: Low-res preview video

### StoryCore-Engine Integration

1. Export project as JSON
2. Open StoryCore-Engine CLI
3. Run generation commands:
   ```bash
   storycore grid --project my-project
   storycore promote --project my-project
   storycore qa --project my-project
   ```

### Saving Projects

- **Auto-save**: Every 2 minutes
- **Manual save**: Ctrl+S or File > Save
- **Save As**: File > Save As (create copy)

## Troubleshooting

### Common Issues

**Problem**: Shots not appearing in canvas
- **Solution**: Check if shots are hidden by filters. Clear search/filter.

**Problem**: Audio not playing
- **Solution**: Check volume levels, ensure track is not muted, verify audio file format.

**Problem**: Drag-and-drop not working
- **Solution**: Ensure you're dragging from the correct area (asset thumbnail). Try refreshing the page.

**Problem**: Export fails
- **Solution**: Check project for errors (missing assets, invalid durations). Verify disk space.

**Problem**: Chat Assistant not responding
- **Solution**: Check internet connection. Verify API credentials in settings.

### Performance Tips

- **Optimize images**: Use compressed formats (JPG instead of PNG)
- **Limit effects**: Too many effects can slow preview
- **Close unused panels**: Hide Asset Library when not needed
- **Clear cache**: Settings > Clear Cache

### Getting Help

- **Documentation**: Help > User Guide

## Best Practices

### Project Organization

1. **Use descriptive names**: "Opening Scene - Sunrise" not "Shot 1"
2. **Group related shots**: Keep scenes together
3. **Consistent duration**: Maintain rhythm (e.g., 5-second shots)
4. **Asset naming**: Use clear, searchable names

### Audio Workflow

1. **Start with dialogue**: Add voiceover/dialogue first
2. **Layer music**: Add background music second
3. **Add SFX**: Sound effects last
4. **Balance levels**: Dialogue loudest, music background, SFX accents
5. **Use automation**: Fade music during dialogue

### Visual Consistency

1. **Color palette**: Stick to 3-5 main colors
2. **Effect style**: Use similar effects across shots
3. **Text style**: Consistent fonts and sizes
4. **Transition style**: Don't mix too many transition types

### Performance

1. **Work in sections**: Complete one scene before moving to next
2. **Save frequently**: Use Ctrl+S often
3. **Preview regularly**: Check work in timeline playback
4. **Export incrementally**: Export sections as you complete them
