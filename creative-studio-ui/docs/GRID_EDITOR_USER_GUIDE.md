## Introduction

L'Éditeur de Grille Avancé est un outil puissant pour organiser et éditer vos plans vidéo. Il offre une interface intuitive avec des fonctionnalités professionnelles pour gérer vos séquences efficacement.

### Fonctionnalités Principales

- **Visualisation vidéo** : Prévisualisez vos plans avec contrôles de lecture
- **Glisser-déposer** : Réorganisez facilement vos plans
- **Snap-to-grid** : Alignement automatique pour une disposition cohérente
- **Annuler/Refaire** : Jusqu'à 50 niveaux d'historique
- **Opérations par lots** : Modifiez plusieurs plans simultanément
- **Recherche avancée** : Trouvez rapidement vos plans
- **Responsive** : S'adapte à toutes les tailles d'écran

## Démarrage Rapide

### Ouvrir l'Éditeur

1. Lancez l'application StoryCore Creative Studio
2. Ouvrez un projet existant ou créez-en un nouveau
3. Cliquez sur l'onglet "Éditeur" dans la navigation principale

### Interface Principale

L'interface est divisée en plusieurs zones :

- **Timeline** : Affiche vos plans en séquence
- **Grille** : Vue en grille pour organiser vos plans
- **Panneau de propriétés** : Édition des métadonnées
- **Barre d'outils** : Accès rapide aux fonctions principales



## Visualisation Vidéo

### Lecteur Vidéo

Le lecteur vidéo intégré vous permet de prévisualiser vos plans directement dans l'éditeur.

#### Contrôles de Base

- **Play/Pause** : Cliquez sur le bouton lecture ou appuyez sur `Espace`
- **Seek** : Cliquez sur la timeline ou utilisez les flèches gauche/droite
- **Volume** : Ajustez avec le curseur de volume

#### Navigation Frame par Frame

- **Flèche droite** : Avancer d'une frame
- **Flèche gauche** : Reculer d'une frame
- **Shift + Flèche** : Avancer/reculer de 10 frames

#### Vitesses de Lecture

Changez la vitesse de lecture pour analyser vos plans :

- **0.25x** : Ralenti extrême
- **0.5x** : Ralenti
- **1x** : Vitesse normale
- **1.5x** : Accéléré
- **2x** : Accéléré rapide

#### Aperçu au Survol

Survolez la timeline avec votre souris pour voir un aperçu thumbnail de la frame correspondante.

### Lecture de Séquence

Sélectionnez plusieurs plans pour les lire en séquence continue :

1. Sélectionnez les plans (Ctrl+Clic ou Shift+Clic)
2. Cliquez sur "Lire la séquence"
3. Les plans se liront automatiquement l'un après l'autre



## Glisser-Déposer

### Déplacer des Plans

1. Cliquez et maintenez sur un plan
2. Déplacez votre souris vers la nouvelle position
3. Relâchez pour déposer

#### Indicateurs Visuels

- **Aperçu semi-transparent** : Montre le plan en cours de déplacement
- **Zone de dépôt verte** : Position valide
- **Icône d'interdiction** : Position invalide
- **Compteur** : Nombre de plans sélectionnés

### Sélection Multiple

Déplacez plusieurs plans simultanément :

- **Ctrl+Clic** : Ajouter/retirer des plans de la sélection
- **Shift+Clic** : Sélectionner une plage de plans
- **Ctrl+A** : Sélectionner tous les plans

### Copier en Déplaçant

Maintenez **Ctrl** pendant le glisser pour copier au lieu de déplacer. Un indicateur "+" apparaîtra sur le curseur.

### Annuler un Glisser

Appuyez sur **Escape** pendant le glisser pour annuler l'opération et restaurer la position initiale.

### Défilement Automatique

Déplacez votre souris vers les bords de l'écran pendant un glisser pour activer le défilement automatique.



## Grille et Alignement

### Activer la Grille

Cliquez sur l'icône de grille dans la barre d'outils ou appuyez sur **G** pour activer/désactiver la grille.

### Snap-to-Grid

Lorsque la grille est activée, les plans s'alignent automatiquement sur les lignes de grille.

#### Désactiver Temporairement

Maintenez **Shift** pendant le déplacement pour désactiver temporairement le snap-to-grid et positionner librement.

### Tailles de Grille

Choisissez parmi plusieurs tailles de grille :

- **8px** : Grille fine pour positionnement précis
- **16px** : Grille standard (par défaut)
- **24px** : Grille moyenne
- **32px** : Grille large

### Guides d'Alignement

Des guides intelligents apparaissent automatiquement lorsque vous alignez des plans :

- **Guides verticaux** : Alignement gauche/centre/droite
- **Guides horizontaux** : Alignement haut/centre/bas
- **Guides de distance** : Espacement égal entre plans

### Distribution Uniforme

Sélectionnez plusieurs plans et cliquez sur "Distribuer uniformément" pour les espacer de manière égale.



## Annuler/Refaire

### Utilisation de Base

- **Annuler** : Ctrl+Z (jusqu'à 50 niveaux)
- **Refaire** : Ctrl+Shift+Z ou Ctrl+Y

### Historique des Actions

L'interface affiche l'action qui sera annulée/refaite :

- "Annuler : Déplacer plan"
- "Refaire : Supprimer plan"

### Indicateur de Modifications

Un point apparaît à côté du nom du projet si des modifications non sauvegardées existent.

### Persistance

L'historique est sauvegardé automatiquement et restauré lors de la prochaine session.

## Menu Contextuel

### Ouvrir le Menu

Cliquez droit sur un plan ou une zone vide pour ouvrir le menu contextuel.

### Actions Disponibles

#### Sur un Plan Unique

- **Dupliquer** : Créer une copie du plan
- **Supprimer** : Retirer le plan (avec confirmation)
- **Exporter** : Exporter le plan
- **Propriétés** : Ouvrir l'éditeur de métadonnées
- **Marquer comme favori** : Ajouter aux favoris

#### Sur Plusieurs Plans

- **Dupliquer la sélection** : Copier tous les plans sélectionnés
- **Supprimer la sélection** : Retirer tous les plans
- **Exporter la sélection** : Exporter en lot
- **Éditer les métadonnées** : Modification groupée

#### Sur Zone Vide

- **Nouveau plan** : Créer un nouveau plan
- **Coller** : Coller depuis le presse-papiers
- **Sélectionner tout** : Sélectionner tous les plans

### Raccourcis Affichés

Le menu affiche les raccourcis clavier associés à chaque action pour un apprentissage rapide.



## Opérations par Lots

### Sélectionner Plusieurs Plans

Utilisez les méthodes de sélection multiple décrites précédemment.

### Barre d'Outils Contextuelle

Une barre d'outils apparaît automatiquement lorsque plusieurs plans sont sélectionnés.

### Opérations Disponibles

#### Dupliquer

Créer des copies de tous les plans sélectionnés avec suffixes numériques uniques.

#### Supprimer

Retirer tous les plans sélectionnés. Une confirmation est demandée si plus de 5 plans sont sélectionnés.

#### Exporter

Exporter tous les plans en parallèle avec barre de progression.

#### Transformer

Appliquer une transformation (rotation, échelle, etc.) à tous les plans.

#### Étiqueter

Ajouter ou modifier des tags sur tous les plans sélectionnés.

### Édition Groupée de Métadonnées

1. Sélectionnez plusieurs plans
2. Cliquez sur "Éditer les métadonnées"
3. Modifiez les champs souhaités
4. Prévisualisez les changements
5. Cliquez sur "Appliquer"

### Progression et Erreurs

- **Barre de progression** : Affiche l'avancement de l'opération
- **Estimation de temps** : Temps restant estimé
- **Rapport d'erreurs** : Liste détaillée des échecs partiels

### Annulation

Cliquez sur "Annuler" pendant une opération pour l'interrompre. Tous les plans affectés seront restaurés.



## Éditeur de Frame

### Ouvrir l'Éditeur

Double-cliquez sur un plan ou sélectionnez "Propriétés" dans le menu contextuel.

### Navigation Frame par Frame

- **Flèches gauche/droite** : Naviguer frame par frame
- **Page Up/Down** : Sauter 10 frames
- **Home/End** : Aller au début/fin

### Édition de Métadonnées

Tous les champs sont éditables avec validation en temps réel :

- **Nom** : Nom du plan
- **Durée** : Durée en secondes ou frames
- **Tags** : Étiquettes séparées par virgules
- **Notes** : Commentaires libres

### Points d'Entrée/Sortie

Définissez les points de montage :

- **I** : Marquer le point d'entrée (In)
- **O** : Marquer le point de sortie (Out)
- La durée est calculée automatiquement

### Zoom et Aperçu

- **Molette de souris** : Zoomer jusqu'à 400%
- **Clic droit + glisser** : Déplacer la vue zoomée
- **Double-clic** : Réinitialiser le zoom

### Comparaison de Frames

Cliquez sur "Comparer" pour afficher deux frames côte à côte :

1. Sélectionnez la première frame
2. Cliquez sur "Marquer pour comparaison"
3. Naviguez vers la seconde frame
4. Utilisez le slider pour comparer

### Formats de Timecode

Choisissez votre format préféré :

- **Frames** : Numéro de frame (ex: 150)
- **Secondes** : Temps en secondes (ex: 5.00s)
- **SMPTE** : Format professionnel (ex: 00:00:05:00)



## Recherche et Filtrage

### Barre de Recherche

Tapez dans la barre de recherche pour filtrer les plans en temps réel (< 200ms).

### Critères de Recherche

Recherchez par :

- **Nom** : Nom du plan
- **Tags** : Étiquettes associées
- **Durée** : Durée du plan
- **Type** : Type de plan (wide, close-up, etc.)
- **Statut** : Statut (brouillon, final, etc.)

### Opérateurs Logiques

Combinez les critères avec des opérateurs :

- **AND** : Les deux critères doivent être vrais
- **OR** : Au moins un critère doit être vrai
- **NOT** : Exclure les résultats correspondants

Exemple : `tag:action AND NOT status:draft`

### Navigation dans les Résultats

- **Flèches haut/bas** : Naviguer entre les résultats
- **Entrée** : Ouvrir le résultat sélectionné
- Le résultat actuel est mis en surbrillance

### Filtres Prédéfinis

Accès rapide aux filtres courants :

- **Favoris** : Plans marqués comme favoris
- **Récents** : Plans modifiés récemment
- **Non utilisés** : Plans jamais exportés
- **Avec erreurs** : Plans avec problèmes de qualité

### Sauvegarder des Filtres

1. Créez votre requête de recherche
2. Cliquez sur "Sauvegarder le filtre"
3. Donnez-lui un nom
4. Accédez-y rapidement depuis le menu déroulant

### Suggestions

Si aucun résultat n'est trouvé, l'interface suggère des alternatives basées sur votre requête.



## Copier/Coller

### Copier des Plans

- **Ctrl+C** : Copier les plans sélectionnés
- **Ctrl+X** : Couper les plans sélectionnés

### Coller des Plans

- **Ctrl+V** : Coller à la position actuelle

### Indicateur de Presse-Papiers

Un indicateur visuel apparaît lorsque le presse-papiers contient des plans, montrant le nombre de plans copiés.

### Métadonnées Préservées

Lors du collage, toutes les métadonnées sont copiées sauf :

- **ID** : Un nouvel ID unique est généré
- **Timestamps** : Date de création/modification mise à jour

### Copier entre Séquences

Vous pouvez copier des plans d'une séquence et les coller dans une autre :

1. Ouvrez la première séquence
2. Copiez les plans (Ctrl+C)
3. Ouvrez la seconde séquence
4. Collez les plans (Ctrl+V)

La compatibilité est vérifiée automatiquement.

## Export et Partage

### Exporter la Configuration

Sauvegardez votre disposition et paramètres :

1. Cliquez sur "Fichier" > "Exporter la configuration"
2. Choisissez le format :
   - **JSON** : Format standard
   - **YAML** : Format lisible
   - **URL** : Lien partageable
3. Sélectionnez ce qui doit être inclus :
   - Disposition de grille
   - Paramètres de snap
   - Préférences visuelles

### Importer une Configuration

1. Cliquez sur "Fichier" > "Importer la configuration"
2. Sélectionnez le fichier ou collez l'URL
3. Résolvez les conflits si nécessaire
4. Cliquez sur "Appliquer"

### Partager via URL

Générez une URL contenant votre configuration :

1. Exportez en format URL
2. Copiez le lien généré
3. Partagez-le avec votre équipe
4. Ils peuvent l'importer directement

### Templates

Créez des templates réutilisables :

1. Configurez votre disposition idéale
2. Exportez comme template
3. Seuls les paramètres réutilisables sont inclus
4. Importez-le dans de nouveaux projets

## Conseils et Astuces

### Optimisation des Performances

Pour les séquences avec 30-50 plans :

- **Utilisez les optimisations React** : L'éditeur utilise automatiquement memo, useMemo et useCallback
- **Chargement asynchrone** : Les thumbnails se chargent en arrière-plan
- **Cache intelligent** : Les aperçus sont mis en cache pour un accès rapide

### Workflow Efficace

1. **Organisez d'abord** : Utilisez la grille pour structurer vos plans
2. **Utilisez les tags** : Facilitez la recherche ultérieure
3. **Sauvegardez régulièrement** : L'auto-save fonctionne toutes les 30 secondes
4. **Exploitez les raccourcis** : Gagnez du temps avec les raccourcis clavier

### Gestion de Projet

- **Nommez clairement** : Utilisez des noms descriptifs pour vos plans
- **Utilisez les favoris** : Marquez les plans importants
- **Créez des templates** : Réutilisez vos configurations préférées
- **Exportez régulièrement** : Sauvegardez vos configurations importantes

### Résolution de Problèmes

#### Les plans ne s'alignent pas

- Vérifiez que la grille est activée (touche `G`)
- Assurez-vous de ne pas maintenir `Shift`

#### La vidéo ne se charge pas

- Vérifiez le format du fichier (MP4, WebM supportés)
- Assurez-vous que le fichier existe et est accessible
- Consultez la console pour les erreurs

#### Les performances sont lentes

- Réduisez le nombre de plans visibles avec les filtres
- Fermez les autres applications gourmandes
- Vérifiez que vous avez moins de 50 plans (cas d'usage optimal)

#### L'historique ne fonctionne pas

- Vérifiez que vous n'avez pas atteint la limite de 50 niveaux
- Assurez-vous que l'action est annulable
- Redémarrez l'application si le problème persiste

### Accessibilité

L'éditeur respecte les préférences système :

- **prefers-reduced-motion** : Réduit les animations
- **Navigation clavier** : Toutes les fonctions accessibles au clavier
- **Contraste élevé** : Support du mode contraste élevé
