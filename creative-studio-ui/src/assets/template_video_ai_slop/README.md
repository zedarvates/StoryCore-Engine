# Template Vidéo AI Slop

Ce dossier contient les fichiers nécessaires pour créer un template vidéo de type AI slop, destiné à YouTube avec placement de produit ou ventes diverses.

## Structure du Dossier

- `animations.js` : Fichier contenant les fonctions pour les animations utilisées dans le template.
- `effets_speciaux.js` : Fichier contenant les fonctions pour les effets spéciaux utilisés dans le template.
- `template_video.js` : Fichier principal pour générer le template vidéo complet.

## Utilisation

### Prérequis

- Node.js installé sur votre machine.
- Un éditeur de code pour modifier les fichiers si nécessaire.

### Étapes pour Utiliser le Template

1. **Cloner le Dossier** :
   ```bash
   git clone <repository-url>
   cd assets/template_video_ai_slop
   ```

2. **Installer les Dépendances** :
   ```bash
   npm install
   ```

3. **Exécuter le Template** :
   ```bash
   node template_video.js
   ```

4. **Personnaliser le Template** :
   - Modifiez les fichiers `animations.js` et `effets_speciaux.js` pour ajuster les animations et effets spéciaux.
   - Modifiez le fichier `template_video.js` pour ajuster la structure du template et les produits présentés.

## Exemple d'Utilisation

Le fichier `template_video.js` contient un exemple d'utilisation avec deux produits. Vous pouvez modifier cet exemple pour inclure vos propres produits et appels à l'action.
