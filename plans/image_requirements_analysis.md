## Technical Specifications

### Image Quality Standards
- **Resolution:** 300 DPI minimum for print materials
- **Color Space:** sRGB for web, CMYK for print
- **File Formats:** PNG for transparency, JPG for photos, SVG for vector icons
- **Optimization:** Web images under 500KB, icons under 100KB

### Consistency Requirements
- **Color Palette:** Primary #1e3a8a (blue), Secondary #7c3aed (purple), Accent #f59e0b (amber)
- **Typography:** Inter or SF Pro Display for headers, Inter for body
- **Style:** Clean, modern, professional, tech-forward
- **Branding:** Film/video production theme with AI/tech elements

### Accessibility Considerations
- **Contrast:** WCAG AA compliant (4.5:1 ratio minimum)
- **Alt Text:** Descriptive filenames and metadata
- **Scalability:** Vector formats where possible for different sizes

## AI Generation Optimization

### Prompt Engineering Best Practices
1. **Be Specific:** Include dimensions, colors, style references
2. **Use References:** Mention existing design systems (Material Design, etc.)
3. **Specify Use Case:** Explain where and how the image will be used
4. **Include Technical Details:** File format, transparency requirements
5. **Consistency Instructions:** Reference brand colors, style guidelines

### Batch Generation Strategy
1. **Icon Sets:** Generate as complete sets for consistency
2. **Variations:** Create multiple options for each critical asset
3. **Refinement:** Iterate based on integration feedback
4. **Quality Control:** Review all images for brand consistency
